Overview

Returns a list of mobile devices that are not in compliance with corporate policies.

Endpoint and URI

https://itsapi.itsupport247.net/repo...s/noncompliant

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Response Body

{

"devices":[

{

"details":"Passcode not required",

"MachineId": 9223372036854775807,

"name":"iPhone_1"

},

{

...

},

{

...

}

]

"error": null

}

Parameter Descriptions for Noncompliant Mobile Devices
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"details"	Reason for noncompliance
"MachineId"	Unique machine ID in ConnectWise database
"name"	Device name in Asio™ Platform (ITSupport Portal)
