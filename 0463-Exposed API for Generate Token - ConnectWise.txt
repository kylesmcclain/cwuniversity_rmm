Overview

The Generate Token API gets a token for a given company and site location.

API Details

The following are the API details that are available within our developer portal for API integration.

Method	API	Description	Parameter Required	Parameter Type	Content-Type


POST

	/api/platform/v1/device/token	Get a token for a given company and site location	Yes – ClientId and SiteId	

string

	application/json
{
"clientID": "<sample public client id>",
"siteID": "<sample public site id>"
}


 

Request Field Descriptions:

Field Name	Description	Type/Format	Example/Sample Value
clientID	A unique identifier of a company	string	cxx6e8f2-13xx-4fxx-bfxx-fd23243xxxxx
siteID	A unique identifier of a site	string	e771xxxx-fexx-42xx-a3xx-9678caf9xxxx

 

Response Field Descriptions:

Field Name

	

Description

	

Type/Format

	

Example/Sample value


token	The token is unique per company and site location, used to map the device to the associated company and site.	string	4a7bxxxx-a9f6-4fxx-b2xx-0x50xxxd2203
