Overview

Returns a summary of operating systems in use and the corresponding machine count.

Endpoint and URI

https://itsapi.itsupport247.net/repo...ystems/summary

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"server"

}

Response Body

{

"error":null,

"operatingSystemSummary":[

{

"count":4,

"operatingSystem":"Win2008"

},{

"count":2,

"operatingSystem":"Win2003"

},

{

...

},

{

...

}

]

}

Parameter Descriptions for Operating System Summary
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"count"	Machine count
"operatingSystem"	OS for the listed machine count
Source of Data in the Asio Platform

Reports > Asio > Executive Report > Server Data / Desktop Data > Operating Systems > Number of Machines
