Overview

Returns the status of antivirus software on each machine for each day in the given month.

Endpoint and URI

https://itsapi.itsupport247.net/repo...ntivirus/daily

Method

POST

Request Body

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"desktop"

}

Parameter Descriptions for Daily Antivirus Status
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"dailyAntivirusStatus"	Daily statuses for the resource for each day of the month
"MachineId"	Unique machine ID in ConnectWise database
"resource"	Resource name
