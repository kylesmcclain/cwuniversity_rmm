User Login Report Details
Overview

This report shows which users have logged in to the Asio platform and for how long.

 To locate this screen, navigate to Reports > Asio > Detailed Reports > Audits > User Login Report.

User Login Report Details

The UserName or Email drop-down menu allows you to select All Users or individual users for a 60-day interval. Select the Deleted User(s) checkbox to view deleted users in your report. You can download the report by clicking the Export To Excel tab.

Note: The date and time of the user logs report in the date and time on the user's local machine.
