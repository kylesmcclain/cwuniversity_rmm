Overview

Retrieves a list of SNMP devices at the given site.

Endpoint and URI

https://itsapi.itsupport247.net/repo...n/snmp/devices

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Response Body

{

"devices":[

{

"friendlyName":"TP-Link_1",

"MachineId": 9223372036854775807,

"name":"TP_1",

"type":"router"

},

{

...

},

{

...

}

]

"error":null

}

Parameter Descriptions for SNMP Devices
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"friendlyName"	Device friendly name in Asio™ Platform (ITSupport Portal)
"MachineId"	Unique machine ID in ConnectWise database
"name"	Device name in Asio™ Platform (ITSupport Portal)
"type"	Device type (router, switch, etc.)

Note: You need to pass the month and the year to get the data for the SNMP devices. These two properties are not optional.

Source of Data in the Asio Platform

Reports >  Asio > Executive Report > SNMP Monitoring > Device Details
