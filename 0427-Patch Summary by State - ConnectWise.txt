Overview

Returns the total number of patches in each state (Installed, Missing, Blacklisted, etc.)

Endpoint and URI

https://itsapi.itsupport247.net/repo.../patches/state

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"server"

}

Response Body

{

"error":null,

"patchSummary":[

{

"blacklisted":17,

"installed":98,

"markedForDeployment":0,

"missing":0,

"others":0,

"underReview":10

}

]

}

Parameter Descriptions for Patch Summary by State
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"blacklisted"	Total number of blacklisted patches
"installed"	Total number of installed patches
"markedForDeployment"	Total number of patches marked for deployment
"missing"	Total number of missing patches
"others"	Total number of other patch types
"underReview"	Total number of patches under review
Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Server Data / Desktop Data > Patch Updates > By State
