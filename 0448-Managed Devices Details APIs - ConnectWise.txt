Overview

These APIs return details of your MSP network – Clients, Sites, and Endpoints.  

This information is used to correctly map organizations, companies, and sites between ConnectWise RMM and the vendor's product. We have 4 different GET APIs: 

Get List of Clients - Returns a list of all the clients for an MSP (Client Name and Client ID).
Get List of Sites - Returns a list of sites for a given client (Site Name and Site ID).
Get List of Endpoints - Returns a list of all the endpoints for a given client and site (Endpoint ID and Endpoint Type).
Get Endpoint Details - Returns details of endpoints for a given Endpoint ID.
Get List of Clients

Returns a list of all the clients for an MSP (Client Name and Client ID).

Request

Sandbox: GET https://openapi.sandbox.use1.cwcmddev.cwnet.io/v1/clients 

Production: GET https://openapi.service.itsupport247.net/v1/clients 

Headers

Authorization:Bearer i9OpjK8IBYYNbV9VGJo6APG224I 

Response

200

Headers

Content-Type:application/json

Body

[

{

"clientName": "NSC",

"clientId": "xWtguxurixSAcxCx",

}

]

Get List of Sites

Returns a list of sites for a given client (Site Name and Site ID).

Request

Sandbox: GET https://openapi.sandbox.use1.cwcmddev.cwnet.io/v1/clients/clientID/sites 

Production: GET https://openapi.service.itsupport247.net/v1/clients/clientID/sites 

Parameters
clientID	Example Client Id: 2lUrtru6uvuVs7 	String
Headers

Authorization:Bearer i9OpjK8IBYYNbV9VGJo6APG224I

Response

200

Headers

Content-Type:application/json

Body

[

{

"siteId": "xWtguxurixSAcxCx",

"siteName": "North Star Coms"

}

]

Get List of Endpoints

Returns a list of all the endpoints for a given client and site (Endpoint ID and Endpoint Type).

Request

Sandbox: GET https://openapi.sandbox.use1.cwcmddev.cwnet.io/v1/clients/clientID/sites/siteID/endpoints 

Production: GET https://openapi.service.itsupport247.net/v1/clients/clientID/sites/siteID/endpoints 

Parameters
clientID	Example clientID: 2lUrtru6uvuVs7 	String
siteID	Example siteID: wOCDc3uZu7upuZ 	String

 

Headers

Authorization:Bearer i9OpjK8IBYYNbV9VGJo6APG224I

Response

200

Headers

Content-Type:application/json

Body

[

{

"endpointID": "4a9caa07-d45a-4c14-aa36-8c1c26e7a029",

"endpointType": "Desktop",

}

]
