Overview

Returns the total number of patches installed and missing in each severity class (Critical, Important, Moderate, Low, etc.)

Endpoint and URI

https://itsapi.itsupport247.net/repo...tches/severity

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"server"

}

Response Body

{

"error":null,

"patchSummary":[

{

"installed":23,

"missing":0,

"severity":"Critical"

},

{

"installed":46,

"missing":0,

"severity":"Important"

},

{

...

},

{

...

}

]

}

Parameter Descriptions for Patch Summary by Severity
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"installed"	Number of installed patches of the specified severity type
"missing"	Number of missing patches of the specified severity type
"severity"	Patch severity type
Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Server Data / Desktop Data > Patch Updates > By Severity
