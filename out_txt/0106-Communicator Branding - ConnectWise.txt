Overview

Under the branding section of your communicator policy, there are features that you can select to enable. Navigate to one of the following locations to view and enable these settings.

Endpoints > Packages > Select Required Policy > Edit

Endpoints > Devices  > Select required endpoint > Settings > Devices > Communicator > Edit > Save
Mouse Over Description
This setting allows you to enter the custom text to be reflected as you Mouse Over Description Text.
If the menu is set to Default, then the default description will be shown.

Taskbar Icon Setting
This setting allows you to select the custom image to be reflected as a Communicator desktop icon.
If the menu is set to Default, then the default icon will be shown.
Communicator Icon Requirements
Communicator Supported file type: bmp or ICO or PNG file
Communicator Supported file size: 32x32 pixels
Communicator depth: 32 bit

Note: For a better visual experience, the below DPI resolutions are recommended for image files:

16x16, 8bit, 72DPI (Recommended)
16x16, 8bit, 96DPI
16x16, 8bit, 144DPI
16x16, 8bit, 300DPI
