Overview

ConnectWise RMM has many out-of-the-box task templates that you can use or duplicate and modify to better serve your automation needs. This page lists the available tasks and information about them.

When running a task, select from the following task categories:

Security
Green Computing
Management
Data Collection
Maintenance

Note: Some tasks apply to multiple categories (i.e., Add local user).

For more information on tasks and task management, visit Tasks Overview.

Note: Microsoft has renamed Azure Active Directory (Azure AD) to Microsoft Entra ID. For details on how this rebranding impacts the following conditions, please see Rebranding Impact section.
