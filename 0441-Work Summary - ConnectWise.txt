Overview

Returns a summary of work performed on all of your desktops and servers.

Endpoint and URI

https://itsapi.itsupport247.net/repo...on/worksummary

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Response Body

{

"error": null,

"workSummary": [

"date": "2016-01-01",

"timeBlock": "30",

"onSite": "n",

Parameter Descriptions for Work Summary
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"date"	Ticket date
"startDate"	Date when work on the ticket began
"endDate"	Date when work on the ticket ended.
"timeBlock"	The time block (in minutes) in which you bill your clients, if applicable.
"onSite"	Whether or not the resource is on-site ("y" or "n")
"description"	Description of the issue
"resolution"	Description of the resolution
Source of Data in Asio Platform

Reports > Asio > Executive Report > Work Status > Summary of Work
