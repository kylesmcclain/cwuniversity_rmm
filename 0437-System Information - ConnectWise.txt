Overview

Retrieves system information gathered from a specified machine, including hardware, OS, and BIOS information.

Endpoint and URI

Parameter Descriptions for System Information
Parameter	Description
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"BIOSManufacturer"	BIOS manufacturer name
"BIOSName"	BIOS name
"BIOSSerialNumber"	BIOS serial number
"BIOSVersion"	BIOS version
"BaseBoardManufacturer"	BaseBoard manufacturer name
"DiskInterfaceType"	Type of disk interface
"DiskManufacturer"	Disk manufacturer name
"DiskModel"	Disk model name
"DiskVolume"	Disk volume (in bytes)
"ExternalIPAddress"	Resource external IP address
"FriendlyName"	Resource friendly name in Asio™ Platform (ITSupport Portal)
"IPAddress"	Resource local IP address
"LastCheckDate"	Last agent check date
"MachineId"	Unique machine ID in ConnectWise database
"MachineName"	Resource name in Asio™ Platform (ITSupport Portal)
"Model"	Resource model name
"OSInstallDate"	OS install date and time
"OperatingSystem"	Resource OS
"Partitions"	Number of partitions
"ProcessorCount"	Number of processors
"ProcessorType"	Type of processors
"Product"	BaseBoard product name
"RAM"	RAM (in bytes)
"SerialNumber"	Resource serial number
"SystemType"	Resource system type
"TimeZone"	Local time zone of resource
"Type"	Desktop or server
