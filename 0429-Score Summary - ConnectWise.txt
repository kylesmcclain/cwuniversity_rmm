Overview

Returns various scoring metrics for a given site.

Endpoint and URI

https://itsapi.itsupport247.net/repo.../summary/score

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Response Body

{

"error":null,

"scores":[

{

"desktopAntivirus":21,

"desktopDiskSpace":93,

"desktopMalware":80,

"desktopPatch":100,

"desktopTempFile":-1,

"desktopWarranty":-1,

"overall":69,

"serverAntivirus":64,

"serverAvailability":59,

"serverCpuUtilization":100,

"serverDiskUtilization":100,

"serverMemoryUtilization":100,

"serverPatch":50,

"serverWarranty":10

}

]

}

Parameter Descriptions for Score Summary*
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"desktopAntivirus"	

These metrics contribute to the overall score provided in Executive Reports. Each metric score ranges from 0 to 100, which may be percentages for utilization metrics or points for non-utilization metrics. The metrics are in turn used to rank and color code corresponding areas of machine health (Antivirus, Disk Space, etc.), ranging from "Upgrade Needed" (red) to "Excellent" (green).

For a detailed description of these metrics and calculations, see the "Explanation of Charts" section at the bottom of any Executive Report, as well as the Executive Reports page.

Note: A value of -1 for any metric indicates that the metric is not applicable at the given site. For example, sites without servers would return a -1 for all server-related metrics.
