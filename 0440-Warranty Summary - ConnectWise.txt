Overview

Returns the count of resources with active and expired warranties, and the count of resources where warranty information is not available.

Endpoint and URI

https://itsapi.itsupport247.net/repo...rranty/summary

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"server"

}

Response Body

{

"active":229,

"dataUnavailable":7,

"expired":0

"error":null,

}

Parameter Descriptions for Warranty Summary
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"active"	Number of resources with active warranties
"dataUnavailable"	Number of resources with unavailable warranty data
"expired"	Number of resources with expired warranties
Source of Data in Asio Platform

Reports > Asio > Executive Report > Server Data / Desktop Data > Warranty > Status
