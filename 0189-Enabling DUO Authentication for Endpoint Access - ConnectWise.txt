Steps to Enable DUO Authentication
Overview

This guide outlines the steps to enable DUO Time-based One-Time Password (TOTP) authentication, allowing the ConnectWise Network Operations Center (NOC) to securely access your endpoints that are protected by DUO. By following this process, you can ensure seamless integration and secure access management.

Steps to Enable DUO Authentication
Log in to your DUO Admin Console and enroll the user/account using your email address. This will generate system credentials.

Generate the activation link for the enrolled user and share it with the ConnectWise NOC team at srt@connectwise.com.

The NOC will use the activation link to generate a Seed HEX Code and return it to you.
In your DUO Admin Console, attach the provided HEX code as a Hardware Token to the system-generated user credentials.

Select Token Type: TOTP 6 Digit
Enter the HEX code and click 'Import Hardware Tokens'
Once the token is imported, the NOC will create an account for this user in ITBoost.
The NOC will use the Seed Base 32-bit/Secret Key mapped to the HEX code to generate TOTP codes.
This completes the setup, enabling the NOC to log in to endpoints using DUO TOTP authentication.
