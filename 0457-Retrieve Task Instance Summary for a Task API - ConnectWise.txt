API Name

Retrieve Task Instance Summary for a Task

API Description

This API allows you to retrieve information about task instances for a task. This represents the History tab for a specific Task on Scheduled Tasks.

Request and Response API

You can find the Request and Response API details at this dev portal.

The API Response will include the following fields:

Field	Description
TaskID	Unique identifier for the task
Id	Unique identifier for the task instance
CreatedOn	Date and time when the task was created
Name	Name or title of the task
OverallStatus	Overall status of the task execution (e.g., SUCCESS, FAILURE)
Regularity	Specifies the regularity of the task (e.g., recurrent, one-time)
NextSchedule	Date and time of the next scheduled execution for the task
ExecutedOn	Date and time when the task was last executed
DeviceCount	Total count of devices associated with the task
SuccessCount	Number of successful executions for the task
FailedCount	Number of failed executions for the task
ScheduledCount	Number of devices scheduled for the task
PostponedCount	Number of devices for which the task execution has been postponed
DisabledCount	Number of devices for which the task is disabled
RunningCount	Number of devices currently running the task
CanceledCount	Number of devices for which the task execution has been canceled
WaitingCount	Number of devices waiting for the task execution
Count	Total count of devices scheduled for the task

 

Note: Review the Scheduled Tasks documentation for more information.
