RMM Scan Configuration Report Details
Overview

The RMM Scan Configuration page displays configuration details for the RMM Scanning Tool based on the Site selected.

 

To locate this screen, navigate to 

Reports > Asio > Detailed Reports > Agents > RMM Scan Configuration.

RMM Scan Configuration Report Details

Select a Site from the Select Site drop-down list. If the Site contains an agent schedule that was configured in the Remote Agent Deployment IP Scanner, the configuration details will appear below the Site Name field.
