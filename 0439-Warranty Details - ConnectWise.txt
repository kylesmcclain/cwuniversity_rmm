Overview

Retrieves warranty information for the authenticated partner and the specified site.

Endpoint and URI

https://itsapi.itsupport247.net/repo...rranty/details

Method

POST

"SiteCode": "AGTSolutions"

Parameter Descriptions for Warranty Details
Parameter	Description
"IncludeWarrantyHistory"	0 or 1 (For no or yes)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Description"	Warranty description
"ExpirationDate"	Warranty expiration date
"FriendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"LastContactDate"	Last agent contact date
"LastUpdateDate"	Date and time of last warranty update
"MachineId"	Unique machine ID in ConnectWise database
"Manufacturer"	Machine manufacturer
"NextUpdateDate"	Date and time of next warranty update
"OperatingSystem"	Machine operating system
"Provider"	Warranty provider
"PurchaseDate"	Date machine was purchased
"ResourceName"	Machine name in Asio™ Platform (ITSupport Portal)
"ResourceType"	Desktop or server
"SerialNumber"	Machine serial number
"ServiceTag"	Machine service tag
"SiteName"	Site name in Asio™ Platform (ITSupport Portal)
