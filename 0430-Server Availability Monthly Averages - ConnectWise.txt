Overview

Retrieves server availability monthly averages.

Endpoint and URI

https://itsapi.itsupport247.net/repo...bility/monthly

Method

POST

Request Body

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Parameter Descriptions for Server Availability Monthly Averages
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"availability"	

Server availability* for the specified month


"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddress"	Machine IP address
"MachineId"	Unique machine ID in ConnectWise database
"serverName"	Server name in Asio™ Platform (ITSupport Portal)

* For details about how server availability is calculated, see the "Overall Score Calculation" table in the Executive Reports page.

Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Server Data > Availability > Monthly Averages Details
