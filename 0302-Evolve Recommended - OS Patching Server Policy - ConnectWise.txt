Overview

This article covers the recommended settings to employ for your OS Patching Server Policy. These best practice settings for different RMM configurations are based on the feedback and analysis of policies used by partners.

Policy Details
Policy Usage	OS Patching Server policies allow the partner to define how the Server endpoints are patched by the RMM Agent.
Policy Type	OS Patching Server
Policy Name	Evolve Recommended - OS Patching Server Policy
Policy Applies To	

These settings will be applied to the following types of agents:

RMM Enabled Windows Server

RMM Enabled Linux Server
