Overview

Returns the monthly minimum, maximum, and average memory utilization, as well as average daily memory utilization for each server.

Parameter Descriptions for Memory Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"average"	Average memory utilization for the specified month
"dailyAverages"	Average memory utilization for each day of the month
"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddresses"	Machine IP address
"MachineId"	Unique machine ID in ConnectWise database
"max"	Maximum memory utilization for the month
"min"	Minimum memory utilization for the month
"server"	Server name
