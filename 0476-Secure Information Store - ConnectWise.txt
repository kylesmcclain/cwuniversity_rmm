Overview

The Secure Information Store is a web-based vault designed to help you securely store and manage sensitive information.

The Secure Information Store runs on an SQL Server database using 256-bit AES encryption at both the database and column level, the same technology used in banks, credit card processors, hospitals, and government agencies. This level of encryption, combined with Operating System and SQL Server security, creates an extremely secure environment. Additionally, the SQL Server is not directly accessible from the internet, which further minimizes the risk of vulnerability.
