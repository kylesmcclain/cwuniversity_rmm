Overview

Returns the monthly minimum, maximum, and average CPU utilization, as well as average daily CPU utilization for each server.

Endpoint and URI

https://itsapi.itsupport247.net/repo...erformance/cpu

Method

Parameter Descriptions for CPU Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"average"	Average CPU utilization
"dailyAverages"	Daily CPU averages
"friendlyName"	Server friendly name in Asio™ Platform (ITSupport Portal)
"iPAddresses"	Server IP address
"MachineId"	Unique machine ID in ConnectWise database
"max"	Maximum CPU utilization
"min"	Minimum CPU utilization
"server"	Server name
Source of Data in the Asio Platform

Reports > Asio > Executive Report > Server Data > Performance Analysis > CPU Utilization
