Overview

Retrieves agent information for the authenticated partner and the specified site.

Endpoint and URI

https://itsapi.itsupport247.net/repo.../agent/details

Method

Parameter Descriptions for Agent Details
Parameter	Description
"SiteCode"	Unique site code in Asio™ Platform (ITSupport Portal)
"FriendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"HasWebroot"	Machine has Webroot installed ("y" or "n")
"IPAddress"	Site IP address
"Id"	Resource ID
"InstallDate"	Agent install date
"IsCurrent"	

Agent is current ("y" or "n")


"IsDuplicate"	Agent is duplicate ("y" or "n")
"IsGateWay"	Agent is gateway ("y" or "n")
"LastBoot"	Last boot date and time
"LastContact"	Last contact date and time
"MachineId"	Unique machine ID in ConnectWise database
"OS"	Operating system
"ResourceName"	Machine name in Asio™ Platform (ITSupport Portal)
"Site"	Site name listed in Asio™ Platform (ITSupport Portal)
"Type"	Resource type
"Version"	Agent version
