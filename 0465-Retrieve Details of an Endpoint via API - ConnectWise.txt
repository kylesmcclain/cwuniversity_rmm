Overview

The Devices APIs retrieve different types of information related to a specific machine or machines belonging to a specific partner or site.

API Details

Below is a list of Devices API that are available within our developer portal for API integration.

API

	

Description

	

Parameter Required

	

Parameter Type




GET

	

/api/platform/v1/device/endpoints

	

Get details of devices at a partner level

	

No

	

 




GET

	

/api/platform/v1/device/clients/{clientID}/endpoints

	

Get details of a devices of a specific company

	

Yes – client ID

	

string




GET

	

/api/platform/v1/device/clients/{clientID}/sites/{siteID}/endpoints

	

Get details of devices of a specific site

	

Yes  - client ID, site ID

	

string
