Overview

Retrieves counts of machines having antivirus protection in the following states: Updated, Outdated, Not Installed, or Information Unavailable.

Endpoint and URI

https://itsapi.itsupport247.net/repo...ivirus/summary

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"Desktop"

}

Response Body

{

"error":null,

"informationUnavailable":4,

"notInstalled":3,

"outdated":0,

"updated":2

}

Parameter Descriptions for Antivirus Summary
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"informationUnavailable"	Number of machines with unavailable antivirus information
"notInstalled"	Number of machines with no antivirus installed
"outdated"	Number of machines with outdated antivirus
"updated"	Number of machines with updated antivirus
Source of Data in the Asio Platform 

Reports > Asio > Executive Report > Desktop Data / Server Data > Antivirus Definitions > Monthly Status
