ScreenConnect Remote Support
Overview

The ScreenConnect® Remote Support feature allows you to access the Remote Support functionality (previously only available via ScreenConnect Portal) via the Asio™ platform. This functionality is the same as available in the ScreenConnect Portal. To understand how ScreenConnect Remote Support works, review the ScreenConnect documentation.

Prerequisites

Below are the prerequisites to ensure ScreenConnect Remote Support launches from the Asio platform:

ConnectWise RMM Premium or Support Add-on entitlements.
Remote Support permission must be enabled from ConnectWise Home > User & Application Settings > Application Roles. Refer to Application Roles for more information.



Note: If you are still unable to launch Remote Support from the left navigation, please refer to this troubleshooting guide.

ScreenConnect Remote Support
After the prerequisites are in place, select Endpoints > Remote Support from the left navigation menu in the Asio platform.


The Remote Support screen provides the same functionality as the ScreenConnect portal. Follow the steps in the Remote support guide to create a session.
