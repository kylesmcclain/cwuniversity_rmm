Overview

Retrieves a list of BCDR Continuity backup appliances provisioned at the given site.

Parameter Descriptions for BCDR Continuity Appliances
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"freeSpace"	Machine free space (in bytes)
"machineCount"	Number of machines
"MachineId"	Unique machine ID in the ConnectWise database
"name"	Machine name
"provisionDate"	Machine provision date and time
"totalOffsiteStorage"	Total off-site storage (in bytes)
"totalSpace"	Total space (in bytes)
