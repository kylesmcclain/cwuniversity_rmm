Parameter Descriptions for Disk Space Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"diskSpace"	Percentage disk space used for the specified resource
"drive"	Local drive containing disk data
"MachineId"	Unique machine ID in ConnectWise database
"resource"	Resource name
Source of Data in the Asio™ Platform
