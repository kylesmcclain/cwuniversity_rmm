Overview

Returns the age of antivirus definitions on desktop machines.

Endpoint and URI

https://itsapi.itsupport247.net/repo...antivirus/ages

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"desktop"

}

Response Body

{

"antivirusAge":[

{

"DaysOld":2,

"MachineId": 9223372036854775807,

"MachineName":"Intel-AMT1"

},

{

...

},

{

...

}

]

"error":null

}

Parameter Descriptions for Antivirus Ages
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop only
"DaysOld"	Antivirus age in days
"MachineId"	Unique machine ID in ConnectWise database
"MachineName"	Machine name listed in Asio™ Platform (ITSupport Portal)
Source of Data in the Asio Platform

Reports > Asio > Executive Report > Desktop Data > Antivirus Definitions > Ages
