Overview

Follow the steps below to set up the integration with HarmonyPSA.

Note: Harmony PSA integration is not available in EU and AU.

Set Up the Integration in the ConnectWise Asio Platform
Open the ConnectWise Asio™ platform.
Navigate to Settings > Account Administration > Integration > PSA Integration.
Select HarmonyPSA from the drop-down menu.

 

Enter the Integrator User and Integrator Password.
Save the API Token for integration within Harmony PSA.
Set Up the Integration in Harmony PSA
Open Harmony PSA.
Navigate to Setup > External Systems > Settings.
Click the plus sign and then select Continuum from the drop-down menu.

 

Fill out the form that has the Harmony-generated ID needed to create the CallbackURL.

Field

	

Description





ID

	

This is generated by Harmony and is used in the Callback URL when setting up PSA integration in Continuum




Name

	

Description of the integration. As this is a Continuum integration I have named this “Continuum” but this can be changed




UserID

	

Your username for Continuum




Password

	

Password for Continuum




TokenID

	

This token is generated in Continuum.  Place this API token here




ImpersonationUserEmail

	

This is the email address of the user within Harmony that has the relevant rights to create this connection.  This will normally be the administrator




ContinuumHelpDeskTeamName

	

Name of the helpdesk team within Continuum




ContinuumNocTeamName

	

Name of the Network team in Continuum

Save the form in Harmony's system.
Create, test, and verify the Callback URL per Harmony's instructions.
After verifying that Callback URL is functional, open the Asio platform and complete the PSA Integration page by clicking Activate.
Follow the rest of the steps in the Harmony documentation for syncing customers and tickets in the Harmony system.
