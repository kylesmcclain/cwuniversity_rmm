Overview

ConnectWise's public ticketing application programming interface (API) allows MSPs or PSA vendors to integrate with our two-way ticket synchronization system to create tickets, update ticket details (add notes, change statuses, etc.), and close tickets from either side of the integration as outlined in the User Acceptance Test. The API supports RESTful protocol and uses a callback mechanism for real-time synchronization. The integration process includes reviewing scope, building the integration using a test-bench, executing user acceptance test cases and receiving authorization, then launching the integration in production.
