Overview

Returns the monthly minimum, maximum, and average pagefile utilization, as well as average daily pagefile utilization for each server.

Endpoint and URI

Parameter Descriptions for PageFile Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"average"	Average pagefile utilization for the specified month
"dailyAverages"	Average pagefile utilization for each day of the month
"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddresses"	Machine IP address
"MachineId"	Unique machine ID in ConnectWise database
"max"	Maximum pagefile utilization for the month
"min"	Minimum pagefile utilization for the month
"server"	Server name
Source of Data in the Asio Platform

Reports > Asio > Executive Report > Server Data > Performance Analysis > Page File (Swap)
