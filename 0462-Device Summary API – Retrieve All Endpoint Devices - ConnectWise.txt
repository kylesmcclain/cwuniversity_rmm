Overview

The Device Summary API retrieves all data from endpoints grouped by company and site for the specified user within the given category. 

Note:

The response is paginated with a maximum of 500 records per page. 
The response has projection support to retrieve only selected fields (max 15).
If no projection fields are specified, only minimal fields (metadata) will be returned by default. Refer to the Projection Guidelines section for more details. 
The Device Summary API is currently limited to the platform category. Refer to the Parameters section for additional details.

API Request Body

This request body is mandatory. The payload must include both resourceType and resources fields.

Validation rules:

If resourceType is company, only 4 values are allowed in the resources array.
If resourceType is site, only 10 values are allowed in the resources array.
If resourceType is endpoint, up to 500 values are allowed in the resources array.

Projection Guidelines

Note: Device Service supports field-based projection. Use the field query parameter with comma-separated values to specify projection, e.g., friendlyName, os, system.If no projection fields are specified, only the default fields will be returned.

 

	

Projectable Fields




Default

	

endpointID, deviceName, friendlyName, resourceType, endpointType




Metadata




endpointID, deviceName, friendlyName, resourceType, endpointType, ipAddress, macAddress, createdAt, createdBy, modifiedAt, modifiedBy.




Categories




platform

	

remoteAddress, type, subResourceType, virtualType, baseBoard, power, bios, drives, physicalMemory, networks, ethernet, firewall, locations, cardReaders, graphicCards, bluetooth, usbs, diagnostic, discBurning, thunderbolts, os, processors, raidControllers, system, keyboards, mouse, monitors, physicalDrives, shares, pciDevices, logicalVolume, antiViruses, rolesFeatures, agentInstalledUTC, createTimeUTC




backup

	

subResourceType, virtualType, os, system, bios, baseBoard, networks

remoteAddress, type, subResourceType, virtualType, os, system, rolesFeatures, agentInstalledUTC, createTimeUTC
