Include User-Defined Fields in Configuration Items
Last updated
Nov 13, 2024
Table of contents
Overview
Overview

After completing migration/synchronization, you need to include the new User-Defined Fields in Autotask.

To add the User-Defined Fields in Autotask, go to Admin > Products and Services > Products > Configuration Item Types and select the checkbox in the Include column to select the User-Defined Fields that are synchronized with each Configuration Item Type.

Please contact Autotask Support if you need help with including User-Defined Fields.

Note: While creating UDF in Autotask PSA, you should not be creating those UDFs in protected state.
