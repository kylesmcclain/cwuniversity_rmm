Overview

The Agent Last Seen API retrieves information about the availability of the endpoint (offline/online). This API supports bulk input of endpoints.

You can find the following sections in this API document:

Fetches information about the availability of the endpoint (online/offline) to determine when the agent was last seen.

API Request Body

This request body is mandatory. The payload must include both resourceType and resources fields.

Validation rules:

If resourceType is endpoint, a maximum of 400 values are allowed.
