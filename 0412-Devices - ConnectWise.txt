Note: This is planned for deprecation.

Parameter Descriptions for Devices
Parameter	Description
"assetType"	Type of device ("Member Server", "Standalone Server", "Member Workstation", etc.)
"companyId"	Unique ID for MSP company in ConnectWise database
"companyName"	MSP company name in Asio™ Platform (ITSupport Portal)
"cpu"	Device CPU
"disks"	Disk details for the device, including drive name and used vs. total available memory (in bytes)
"friendlyName"	Device friendly name in Asio™ Platform (ITSupport Portal)
"hostName"	Device hostname
"ipAddresses"	Device IP address(es)
"lastLoginUser"	Last user of the device
"lastReboot"	Date and time of last reboot for the device
"lastSeenOnline"	Date and time last seen online by ConnectWise Data Center
"lastUpdated"	Date and time of last device update
"macAddresses"	Device Mac address(es)
"machineID"	Unique machine ID in ConnectWise database
"machineName"	Machine name listed in Asio™ Platform (ITSupport Portal)
"manufacturerName"	Device manufacturer name
"memoryTotal"	Total device memory
"modelNumber"	Device model
"modelSKU"	Device model SKU
"operatingSystem"	Device OS
"siteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"siteId"	Unique site ID in ConnectWise database
"siteName"	Site name in Asio™ Platform (ITSupport Portal)
"supportSerialNumber"	Serial number for device support
