Overview

Returns the monthly minimum, maximum, and average disk space utilization, as well as average daily disk space utilization for each server.

Endpoint and URI

Parameter Descriptions for Disk Usage
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"average"	Average disk usage for the specified month
"dailyAverages"	Average disk usage for each day of the month
"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddresses"	Machine IP address
"MachineId"	Unique machine ID in ConnectWise database
"max"	Maximum disk usage for the month
"min"	Minimum disk usage for the month
"server"	Server name
Source of Data in the Asio Platform
