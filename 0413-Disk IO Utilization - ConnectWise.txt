Overview

Returns the monthly minimum, maximum, and average disk IO utilization, as well as average daily disk utilization for each server.

Endpoint and URI

https://itsapi.itsupport247.net/repo...iskutilization

Method

POST

"SiteCode":"AGTSolutions",

Parameter Descriptions for Disk Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"average"	Average disk IO latency for the specified month
"dailyAverages"	Average disk IO latency for each day of the month
"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddresses"	Machine IP address
"MachineId"	Unique machine ID in ConnectWise database
"max"	Maximum disk IO for the month
"min"	Minimum disk IO for the month
"server"	Server name
Source of Data in the Asio Platform
