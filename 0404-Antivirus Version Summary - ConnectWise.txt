Overview

Retrieves currently detected antivirus software and the definition version.

Endpoint and URI

https://itsapi.itsupport247.net/repo...virus/versions

Method

POST

Request Body

{

"SiteCode": "AGTSolutions"

}

Parameter Descriptions for Antivirus Version Summary
Parameter	Description
"SiteCode"	Unique site code in Asio™ Platform (ITSupport Portal)
"AgentType"	Desktop agent or Server agent
"Antivirus"	Antivirus status
"CurrentVersion"	Current antivirus version
"Date"	Date antivirus was last checked
"FriendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"InstalledVersion"	Installed version of antivirus
"MachineId"	Unique machine ID in ConnectWise database
"MachineName"	Machine name in Asio™ Platform (ITSupport Portal)
"OperatingSystem"	Machine operating system
Source of Data in the Asio Platform
