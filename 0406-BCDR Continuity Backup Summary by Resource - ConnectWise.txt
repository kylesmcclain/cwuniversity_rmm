Parameter Descriptions for BCDR Continuity Backup Summary by Resource
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"appliance"	Appliance name
"backupInterval"	Configured interval for backup
"backupsPerDay"	Number of configured backups per day
"MachineId"	Unique machine ID in ConnectWise database
"name"	Device name
"percentFailed"	Percent of failed backups in the month and year requested
"percentOther"	Percent of backups neither failed nor succeeded in the month and year requested
"percentSucceeded"	Percent of successful backups in the month and year requested
Source of Data in the Asio Platform

Reports > Asio > Executive Report > Recover Continuity Backup Data > Backup Status > Per Resource
