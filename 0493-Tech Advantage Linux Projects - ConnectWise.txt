Overview

Linux projects are undertaken as Custom Desktop or Server projects. To request a Linux project, send a project request to the Desktop Projects (DesktopProjects@connectwise.com) or Server Projects (ServerProjects@connectwise.com) team with your project requirements. A price quote will be provided once the team reviews the scope of your project.

Supported Linux Distributions
Ubuntu 8.04 or later
Red Hat Enterprise Linux 5.3 or later
CentOS 5 or later
Fedora release 16 or later
