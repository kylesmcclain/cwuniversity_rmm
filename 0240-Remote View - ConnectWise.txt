Overview

This article provides information about the Remote View functionality in the Asio® platform. Similar to Remote Support, it is now available in Asio.

Permissions

To access this functionality, assign the following permissions from ConnectWise Home.

Remote View

Note: In the ScreenConnect portal, this functionality is available as ScreenConnect View™. However, in Asio, this functionality is available to partners who already have View in ScreenConnect.

How to Access Remote View
Log in to Asio.
Navigate to the top right navigation bar and click on the Remote View option.

After you click Remote View, the following screen displays:


All the features applicable to ScreenConnect View™ are available by default in Asio.
