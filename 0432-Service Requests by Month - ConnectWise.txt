Overview

Returns counts of service requests that have been opened and/or closed over the last 6 months.

Endpoint and URI

https://itsapi.itsupport247.net/repo...quests/monthly

Method

POST

Request Body

{

"Month":4,

"SiteCode":"aAGTSolutions",

"Year":2015,

}

Response Body

{

"error":null,

"serviceRequestSummary":[

{

"Month":"2015-04",

"closed":3,

"opened":4

},

{

...

},

{

...

}

]

}

Parameter Descriptions for Service Requests by Month
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year


"Month"

	1 of the 6 previous months shown
"closed"	Number of closed requests in the specified month
"opened"	Number of opened requests in the specified month
Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Service Request Data > Open / Closed > By Month
