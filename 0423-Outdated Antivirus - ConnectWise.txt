Overview

Returns a list of resources that have outdated antivirus definitions.

Endpoint and URI

https://itsapi.itsupport247.net/repo...virus/outdated

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"server"

}

Response Body

{

"error":null,

"outdatedAntivirus":[

{

"friendlyName":"Windows Server 2008_1",

"iPAddress":"1.1.1.1",

"lastScanDate":"2015-01-01",

"MachineId": 9223372036854775807,

"resourceName":"WS2008_1"

},

{

...

},

{

...

}

]

}

Parameter Descriptions for Outdated Antivirus
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddress"	Machine IP address
"lastScanDate"	Last antivirus scan date
"MachineId"	Unique machine ID in ConnectWise database
"resourceName"	Machine name in Asio™ Platform (ITSupport Portal)
Source of Data in the Asio Platform

Reports > Asio > Executive Report > Server Data / Desktop Data > Antivirus Definitions > Outdated
