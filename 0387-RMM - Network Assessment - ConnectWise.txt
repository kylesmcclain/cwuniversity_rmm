Overview

The Network Assessment tool helps gather information on the machines, software, and user accounts of any IT Network infrastructure. It is composed of the Network Data Collector and the Report Builder. The noninvasive tool provides assets for lead generation; brandable reports are created quickly, and users have unlimited use of the tool when purchased. The following graphic outlines the assessment process, while the topics in this chapter provide more detail.

Network Assessment is provided in partnership with the Network Detective by RapidFire Tools, Inc. For additional descriptions and instructions beyond what is included in this chapter, see RapidFire's Network_Detective_User_Guide.pdf.
