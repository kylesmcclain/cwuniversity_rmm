Overview

Retrieves a list of resources being backed up by BCDR Continuity appliances at the given site.

Endpoint and URI

https://itsapi.itsupport247.net/repo...y247/resources

Method

POST

Request Body

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Parameter Descriptions for BCDR Continuity Protected Resources
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"appliance"	Appliance name
"lastBackupDate"	Last date of backup
"lastBackupStatus"	Last reported backup status
"lastOffsiteBackupDate"	Last date of off-site backup
"localSpaceUsed"	Local space used (in bytes)
"MachineId"	Unique machine ID in ConnectWise database
"name"	Machine name
"offsiteSpaceUsed"	Off-site space used (in bytes)
"volumes"	Protected volumes backed up by the appliance for the machine
Source of Data in the Asio Platform 

Reports > Asio > Executive Report > Recover Continuity Backup Data > Protected Resources
