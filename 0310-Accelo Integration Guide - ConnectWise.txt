Understanding What Syncs with ConnectWise
Overview

You can adopt a fully synchronized workflow by combining the ConnectWise platform with Accelo business tools to manage all client work from one centralized system. Accelo automation eliminates busy work and enables you to focus on higher-level client service.

Accelo integrates with ConnectWise to sync tickets created from ConnectWise alert processes, allowing you to track work progress and time logged as well as bill clients using the Accelo ticket tracking and billing module. The integration syncs client details, ticket type, status, device, and assignment (to either the ConnectWise Network Operations Center [NOC] or the Help Desk Service). Tickets are pushed in real time to the support team for immediate triage while ensuring you have full visibility.

Use this guide to walk through the setup process for the ConnectWise/Accelo integration.

Note: 

Enabling a PSA integration disables email notifications through Notification Rules.
Accelo integration is not available in EU and AU.
1. Prepare Accelo for ConnectWise Tickets

Note: You will need to create two additional Professional Users on your account: one for your ConnectWise NOC and one for your ConnectWise Help Desk. For directions on how to set up additional users, see the Accelo documentation on Creating Users.

In your Accelo account:

Create a Professional User with the first name NOC and the last name Team.
Create a Professional User with the first name ConnectWise and the last name Helpdesk. (These usernames do not have to match exactly; you will map them after you connect.)
Create users in Accelo for all ConnectWise users.
Create appropriate statuses in Accelo to map between ConnectWise and Accelo, including status progressions. You do not need to create every status, and you can map a single Accelo status to multiple ConnectWise statuses.
2. Connect ConnectWise to Accelo

Before you connect to ConnectWise, read through the Accelo documentation on Integration Requirements.

To connect your ConnectWise account:

Log in to the ConnectWise Asio™ platform (ITSupport Portal) (control.itsupport247.net) with Super Admin credentials.
Go to Settings > Account Administration > Integration > PSA Integration.
Select Accelo from the drop-down.
Enter the following information to set up your connection:
Integrator User – Enter your deployment name. For example, if your deployment is mycompany.accelo.com, your username should be mycompany.
Integrator Password – Select a 16-character password that contains numbers, letters, and special characters.
Callback URL
Callback Method – Set as POST.
Copy the API Token for use in the Accelo integration.
Click Activate.

3. Complete the Integration in Accelo
After you have completed the PSA integration, log in to Accelo and navigate to the Integration configuration.
Click your User Profile Icon in the Navigation Bar and select Integrations.
Select the ConnectWise tab.
Enter your ConnectWise credentials:
ConnectWise Integrator Username – Must match the Integrator User in ConnectWise; should be your deployment name.
ConnectWise Integrator Password – Must match the 16-character password entered in ConnectWise.
API Token – Enter the API token generated by ConnectWise.
Click Save.
4. Configure ConnectWise Options

Review the extra configuration options to fully customize your ConnectWise integration.

Under the ConnectWise integrations page, click Options:

Click Link Issue Statuses. For each ConnectWise status, select the corresponding Accelo status for tickets. Create more statuses if required.
Select the ConnectWise status that should be used when pushing data from Accelo to ConnectWise for each Accelo status.
Return to the Integration Screen.
Click Link Users. Link each ConnectWise user to an Accelo user. Do not use the NOC and Help Desk Service users created earlier.
Return to the Integration Screen.
Select the NOC Team user created earlier in the Select the user to be mapped to NOC Team section.
Select the Help Desk Service user created earlier in the Select the user to be mapped to ConnectWise Help Desk Service section.
Select a ConnectWise Ticket type to be used as the ticket type for issues created in Accelo then pushed to ConnectWise.
Select Import to Import all ConnectWise Ticket priorities. This step pulls in the existing ConnectWise priorities and links them to existing Accelo statuses. No additional Accelo statuses will be created.
5. Maximize the ConnectWise Integration

Note: ConnectWise ticket priorities must be imported before Accelo issues are pushed to ConnectWise. Issues created in Accelo are only pushed if assigned to either the NOC or Help Desk Service.

Understanding What Syncs with ConnectWise

All tickets created in your ConnectWise account can be pulled into Accelo. ConnectWise tickets can be pulled into your account if created against a site that matches an Accelo company, but these tickets cannot sync with Accelo since they cannot be assigned against an Accelo company. See the Accelo documentation Create Companies and Contacts for more information.

ConnectWise Field	Where it appears in Accelo's Ticket/Issue
Status	Status
Assign To	Assignee
Priority	Priority
Description	Description

Note: The description field is editable in Accelo. If you update the ticket description in Accelo, the updates sync to the comments section in ConnectWise.


Comments	Saved as an Activity in the ticket/issue stream
Notifications	WILL NOT CAPTURE

 

If you have any questions or need additional support related to your integration with ConnectWise, visit Accelo support.
