Overview
Overview

A critical component in the partner onboarding process, the Secure Information Store allows:

Secure transfer of server credentials to the NOC back office.
Secure transfer of desktop credentials to the Help Desk.
Secure storage of critical client information for access by onsite technicians.
Secure storage of Lights Out Management credentials for remotely re-starting downed servers.
Secure storage of Vault encryption pass-phrases for decrypting and troubleshooting Vault backups.

Note: A primary function of the Secure Information Store is to provide a secure method for transferring administrator credentials to your respective NOC support teams. Secure storing eliminates managed service ticket delays that result from incorrect or non-existent credentials and also eliminates security risks that can occur when passwords are exchanged manually.

The Secure Information Store can also store:

Active Directory credentials
Exchange credentials
Custom Application credentials
SQL credentials
VMware/vCenter credentials
Citrix/Terminal Server credentials

To enter credentials and other login information into the Secure Information Store, use the site-level Passwords tab, or Password Vault, in Settings > Setup > RMM > RMM Setup > Sites Using RMM > [site name].
