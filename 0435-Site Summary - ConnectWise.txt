Overview

Asset and service request summary for a specified site.

Endpoint and URI

https://itsapi.itsupport247.net/repo...n/summary/site

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Response Body

{

"antivirusMachineCount":25,

"antivirusMachineUpdated":5,

"desktopCount":15,

"error":null,

"firewallCount":0,

"malwareThreatsRemoved":5,

"mobileDeviceCount":0,

"patchesInstalled":1998,

"patchesUnderReview":812,

"serverCount":10,

"serviceTicketsClosed":1,

"serviceTicketsOpened":0,

"switchCount":0,

"temporaryFilesRemoved":"0.0 MB",

"thirdPartyPatchesInstalled":178

}

Parameter Descriptions for Site Summary
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"antivirusMachineCount"	Total machines with antivirus
"antivirusMachineUpdated"	Total machines with updated antivirus
"desktopCount"	Total desktop count
"firewallCount"	Total firewall count
"malwareThreatsRemoved"	Total malware threats removed
"mobileDeviceCount"	Total device count
"patchesInstalled"	Total patches installed
"patchesUnderReview"	Total patches under review
"serverCount"	Total server count
"serviceTicketsClosed"	Total service tickets closed
"serviceTicketsOpened"	Total service tickets closed
"switchCount"	Total switch count
"temporaryFilesRemoved"	Total temporary files removed
"thirdPartyPatchesInstalled"	Total third-party patches installed
