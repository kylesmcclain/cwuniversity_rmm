How to Change a Server Password

To change an existing server password:

Log in to the Asio platform.
Navigate to Settings > Setup > RMM.
Click the RMM Setup tab and navigate to Sites Using RMM > select Site.
Click the Passwords tab.

Expand the Server Passwords section.

Click Access Control.

Click the Change Password checkbox.

Create a new Password. Enter it a second time in the Confirm Password field.
Click Save.
