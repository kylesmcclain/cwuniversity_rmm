Overview

This article covers the recommended settings to employ for your RMM Site Configuration Policy. These best practice settings for different RMM configurations are based on the feedback and analysis of policies used by partners.

Policy Details
Policy Usage	RMM policies allow the partner to set the RMM functionality at the site level. System will deploy the appropriate agent components on the endpoints once the site is enabled for RMM.
Policy Type	RMM Site Configuration
Policy Name	Evolve Recommended - RMM Configuration
Policy Applies To	These settings will be applied to a Site.
