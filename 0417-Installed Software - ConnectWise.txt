Overview

Lists the software installed on a specified machine including publisher, version, and installation date.

Endpoint and URI

https://itsapi.itsupport247.net/repo...ware/installed

Method

POST

Request Body

{

"SiteCode":"AGTSolutions"

}

Response Body

{

"Error": null,

"InstalledSoftware": [

{

"Application": "Windows Media Player",

"DateChecked": "2015-08-27",

"FriendlyName": "Mike's PC",

"IPAddress": "1.1.1.1",

"InstallDate": "2015-01-16",

"MachineId": 9223372036854775807,

"MachineName": "Intel-AMT1",

"OperatingSystem": "Microsoft Windows 8.1 Pro",

"Publisher": "Unavailable",

"Version": "9.0"

}

]

}

 

Parameter Descriptions for Installed Software
Parameter	Description
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Application"	Software application name
"DateChecked"	Last date checked by RMM agent
"FriendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"IPAddress"	Machine IP address
"InstallDate"	Application install date
"MachineId"	Unique machine ID in ConnectWise database
"MachineName"	Machine name
"OperatingSystem"	Machine OS
"Publisher"	Application publisher
"Version"	Application version
