Overview

This pod shows the list of all configured devices related to the service ticket and pulls device information in realtime from ConnectWise RMM. Select a device from the list for troubleshooting-related information about that device. This information is displayed in the Device Details section.

Important: This feature is currently available for SSO Migrated Partners.

Column Title	Description


Name

	

Displays the name of the device (Configuration item).




Last User

	

Displays the last user to access the device.




Type

	

Displays the type of device.




IP Address

	

Displays the Internal IP address associated with the device. 




Tickets

	

Displays the number of open tickets associated with the device. Click to open the service ticket list. 




Actions

	

Control Icon

	

Click to select one of the following actions:

ScreenConnect Remote Access 
ScreenConnect Dashboard
ScreenConnect Backstage



Actions Menu

	

Click to select one of the following actions:

Device Summary (Navigates to Asio Portal)
Delete (Remove device from Pod in ticket)

Requirements for Devices Pod 

To use this pod you must have: 

ConnectWise RMM
ConnectWise SSO (ConnectWise Home) enabled for authentication to ConnectWise RMM™ and ConnectWise PSA™. Also logged in from the same user to view the information on the Device Pod.
Have configured integration between PSA and RMM as well as enabled syncing of assets and tickets between RMM and PSA.
ConnectWise PSA version 2022.2 or above
Technician Efficiency Pod Checkbox

We are providing the option for you to Opt In or Opt Out for Technician Efficiency. There will be a checkbox visible on the Site mapping page under the PSA Integration wizard. If you click on the checkbox and finish the wizard, the technician efficiency pod will be visible on the CW PSA Ticketing page. If you want to opt out you can then uncheck the checkbox and complete the wizard again.
