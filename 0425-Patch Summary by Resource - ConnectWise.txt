Overview

Returns the total number of patches in each state (Installed, Missing, Blacklisted, etc.) for each resource of the given resource type.

Endpoint and URI

https://itsapi.itsupport247.net/repo...ches/resources

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"server"

}

Response Body

{

"error":null,

"patchSummary":[

{

"blacklisted":0,

"installed":16,

"MachineId": 9223372036854775807,

"markedForDeployment":2,

"missing":0,

"others":0,

"underReview":1

},

Parameter Descriptions for Patch Summary by Resource
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"blacklisted"	Number of blacklisted patches on the machine
"installed"	Number of installed patches on the machine
"MachineId"	Unique machine ID in ConnectWise database
"markedForDeployment"	Number of patches marked for deployment on the machine
"missing"	Number of missing patches on the machine
"others"	Number of other patch types on the resource
"resource"	Machine name in Asio™ Platform (ITSupport Portal)
"underReview"	Number of patches under review on the resource
Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Server Data / Desktop Data > Patch Updates > By State Per Server / By State Per Desktop
