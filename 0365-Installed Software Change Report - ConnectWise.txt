Installed Software Change Report Details
Overview

This report displays applications that have been installed or removed in a particular date range.

 To locate this screen, navigate to Reports > Asio > Detailed Reports > Technical Details > Installed Software Change Report.

Installed Software Change Report Details
The date range must be from the past 12 months.
The date range can be a maximum of 60 days or fewer.
