Overview

The Asset Change Report contains data for asset changes over a given period. For example, these reports can help you troubleshoot issues end users raise, especially if their systems are running slow or abnormal. You can see physical changes to a machine by running the Logical Memory, Physical Disk, or Adapter Configuration Change Report. If an end user's system is running slow, you may want to check if RAM has been removed from the system or if unauthorized programs have been added (these programs may load modules into RAM on start-up, causing them to run faster when used but slowing down the system as a whole).

  To locate this screen, navigate to Reports > Asio > Detailed Reports > Technical Details > Asset Change Report.

Asset Change Report Details

To access the report:

Select a site.
Select an Agent type (e.g., Desktop, Server, VMware).
Select Agents.
Select a date range.
Select asset change report (e.g., Logical Drive, Logical Memory, Software, Physical Disk).
