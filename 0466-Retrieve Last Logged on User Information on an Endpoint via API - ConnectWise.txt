Overview

The Devices APIs retrieve different types of information related to a specific machine or machines belonging to a specific partner or site.

API Details

The following are the Devices API that are available within our developer portal for API integration.

GET

	

/api/platform/v1/device/endpoints/{endpointId}/system-state-info

	

Get details of logged on users along with time stamp for a specific device

	

Yes – endpoint ID
