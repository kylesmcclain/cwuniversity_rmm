Overview

Returns average daily availability for all servers at the given site.

Endpoint and URI

https://itsapi.itsupport247.net/repo...lability/daily

Method

POST

"SiteCode":"AGTSolutions",

Parameter Descriptions for Daily Server Availability
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"averageDailyAvailability"	Percentage of server availability*
"MachineId"	Unique machine ID in ConnectWise database
"server"	Server name

* For details about how server availability is calculated, see the "Overall Score Calculation" table in the Executive Reports page.

Source of Data in the Asio Platform

Reports > Asio > Executive Report > Server Data > Availability > Daily Averages
