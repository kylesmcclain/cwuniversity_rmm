Overview

Returns a list of resources with unsupported antivirus software installed.

Endpoint and URI

https://itsapi.itsupport247.net/repo...us/unsupported

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

"ResourceType":"desktop"

}

Response Body

{

"error":null,

"unsupportedAntivirus":[

{

"antivirus":"Not Available",

"friendlyName":"Mike's PC",

"iPAddress":"10.2.43.3",

"lastScanDate":"2015-01-29",

"MachineId": 9223372036854775807,

"resourceName":"Intel-AMT1"

},

{

...

},

{

...

}

]

}

Parameter Descriptions for Unsupported Antivirus
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"ResourceType"	Desktop or server
"antivirus"	Antivirus status
"friendlyName"	Machine friendly name in Asio™ Platform (ITSupport Portal)
"iPAddress"	Machine IP address
"lastScanDate"	Last antivirus scan date
"MachineId"	Unique machine ID in ConnectWise database
"resourceName"	Machine name in Asio™ Platform (ITSupport Portal)
