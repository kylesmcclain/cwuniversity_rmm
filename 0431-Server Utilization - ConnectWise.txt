Overview

Retrieves average server utilization data for a given month.

Endpoint and URI

https://itsapi.itsupport247.net/repo...ization/server

Method

POST

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

Parameter Descriptions for Server Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"cPU"	Server CPU usage for the month
"diskSpace"	Server disk space usage for the month
"MachineId"	Unique machine ID in ConnectWise database
"memory"	Server memory usage for the month
"server"	Server name in Asio™ Platform (ITSupport Portal)
Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Server Data > Utilization > Per Server
