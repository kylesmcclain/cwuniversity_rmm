Parameter Descriptions for Daily Server Utilization
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"cPU"	CPU usage for the resource for each day of the month
"diskIO"	Disk IO latency for the resource for each day of the month
"diskUsage"	Disk usage for the resource for each day of the month
"MachineId"	Unique machine ID in ConnectWise database
"memory"	Memory usage for the resource for each day of the month
"pageFile"	PageFile usage for the resource for each day of the month
"server"	Server name
