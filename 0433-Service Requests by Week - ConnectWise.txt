Overview

Returns counts of service requests that have been opened and/or closed over the previous 6 weeks.

Endpoint and URI

https://itsapi.itsupport247.net/repo...equests/weekly

Method

POST

Request Body

{

"Month":4,

"SiteCode":"AGTSolutions",

"Year":2015,

}

Response Body

{

"error":null,

"serviceRequestSummary":[

{

"startDate":"Mar 20"

"endDate":"Mar 26",

"closed":5,

"opened":3,

},

{

...

},

{

...

}

]

}

Parameter Descriptions for Service Requests by Week
Parameter	Description
"Month"	Requested month (numerical)
"SiteCode"	Unique site code listed in Asio™ Platform (ITSupport Portal)
"Year"	Requested year
"startDate"	Start date for the week


"endDate"

	End date for the week
"closed"	Number of closed requests in the specified week
"opened"	Number of opened requests in the specified week
Source of Data in the Asio Platform

Reports >  Asio > Executive Report > Service Request Data > Open / Closed > By Week
