API Name

Retrieve Task Instance Execution Results

API Description

This API allows you to retrieve the task results for a specific task instance. This represents clicking on the ‘+’  icon on any row on the History tab under Scheduled Tasks.

Request and Response API

You can find the Request and Response API details at this dev portal.

The API Response will include the following fields:

Field	Description
taskId	Unique identifier for the task
taskInstanceId	Unique identifier for the specific instance of the task
endpointId	Unique identifier for the endpoint associated with the task
executionTime	Date and time when the task was executed
createdOn	Date and time when the task instance was created
errorDetails	Details of any errors encountered during the task execution
resultDetails	Additional details or information about the task result
completionStatus	Status indicating whether the task execution was successful or failed (e.g., SUCCESS, FAILED)
executorType	Type of executor responsible for executing the task (e.g., PARTNER)
count	Total count of devices scheduled for the task

Note: Review the Scheduled Tasks documentation for more information.
