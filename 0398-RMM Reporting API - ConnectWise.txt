Overview

The ConnectWise RMM Reporting API provides access to more than 200 data fields ranging from basic machine information to patching, endpoint protection, and backup data, allowing you to use raw reporting data to build custom reports in-house or through third-party vendors. Vendor integration opportunities with this open API are expansive, from reporting tools to documentation systems to lead generation. Our current integrations include BrightGauge and IT Glue™.

This API supports REST protocol, uses JSON format for data exchange, and currently responds mostly to HTTP POST requests. A future Reporting API will adhere to common REST standards such as HTTP GET for resource retrieval. Additional reporting data exposed in future releases will be integrated as new resources in future updates to this API.
