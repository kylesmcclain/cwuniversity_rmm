Overview

All alerts will be generated from the Asio platform. In the current release, there is only one Alert, Network Element Offline.

You will receive alerts from ConnectWise when a network infrastructure device goes offline.

Editing a Network Monitoring Alert.

You can edit a Network Monitoring Alert to better fit your needs. Some options include changing the following:

Alert name
Severity
Entities
Trigger conditions
Clear conditions
Notification channels
How to edit an alert
In the Asio Platform Navigate to Endpoints > Alerts > Intelligent Alerts.
Using the search and filter options, locate the alert you need.

Once you have the name of the alert you need, navigate to Endpoints > Network. Click manage to open the Auvik Portal.

Under Admin > Manage Alerts, select Alerts.
Search for the Alert that you want to edit.

Select the Alert and click edit.

Edit the Alert per your needs
